COURSE TITLE

Object-Oriented Programming

CS-203

Instructor

Umur Inan, M.S

Contact Info

Umur Inan Assistant Professor of Computer Science Email: uinan@miu.edu

April 2021

CS203 focuses on Object Oriented Programming paradigm. There are lots of concepts to be grabbed to understand the paradigm itself and course will build on knowledge and skills already gained from CS201, which is the pre-requisite for this course. Along with understanding the paradigm, will also gain enough hands-on experience with coding during labs, assignments and project. We will use Java as our reference programming language throughout the course.

Lesson Topics

  • Write OO Java Code
  • Learn Analysis and Design techniques
  • Understand the OO approach
  • Be able to write GUI programs
  • Understand the role of Creative Intelligence in programming

    • Main Objectives of Object-Oriented Programming

    This course covers programming in Java, specifically focusing on object-oriented concepts and creating GUI applications. Topics include classes and objects, primitives and references, inheritance and polymorphism, interfaces and abstract classes, exception handling, GUI programming in Swing, and serialization and file I/O.
